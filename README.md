# Sistema de Controle de Estoque (Django)

Este projeto foi desenvolvido como parte das atividades do **programa CAPACITA**, promovido pelo **CEPEDI**, durante o curso de **Back-End em Python**.

O objetivo principal deste projeto é demonstrar, de forma prática, a construção de um **CRUD completo utilizando o framework Django**, além da criação de uma interface web simples para interação com o sistema.

## Sobre o projeto

O sistema consiste em uma aplicação web básica de **controle de estoque**, permitindo o gerenciamento de:

* Produtos
* Fornecedores

A aplicação permite realizar as operações fundamentais de um CRUD:

* Criar registros
* Visualizar registros
* Editar registros
* Excluir registros

Além do backend desenvolvido em **Django**, também foi construída uma interface simples utilizando **HTML, TailwindCSS e templates do Django**, permitindo que o usuário interaja com o sistema de forma visual.

## Problema que o sistema resolve

Muitos pequenos comércios ou projetos simples não possuem um sistema para controlar produtos e fornecedores. Normalmente esse controle é feito em anotações ou planilhas, o que pode gerar desorganização e erros.

Este sistema foi desenvolvido para resolver esse problema de forma simples, permitindo cadastrar, visualizar, editar e excluir produtos e fornecedores através de uma interface web. Assim, as informações ficam centralizadas e mais fáceis de gerenciar.


## Tecnologias utilizadas

* Python
* Django
* TailwindCSS
* PostgreSQL
* Node.js (utilizado para integração com Tailwind)
* Gunicorn
* Render (deploy da aplicação)

## Funcionalidades

* Cadastro de produtos
* Edição de produtos
* Exclusão de produtos
* Cadastro de fornecedores
* Edição de fornecedores
* Exclusão de fornecedores
* Relacionamento entre produtos e fornecedores

## Deploy

A aplicação foi publicada utilizando a plataforma **Render**, com banco de dados **PostgreSQL** em ambiente de produção.

## Objetivo do projeto

Este projeto tem caráter **educacional**, sendo desenvolvido para consolidar os conhecimentos adquiridos durante o curso de **Back-End em Python** do programa **CAPACITA (CEPEDI)**.

O foco principal foi entender e aplicar conceitos como:

* Estrutura de projetos Django
* Criação de modelos e banco de dados
* Implementação de CRUD
* Templates e renderização de páginas
* Integração com banco de dados em produção
* Deploy de aplicação web
